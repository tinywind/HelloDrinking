git : 분산버젼관리시스템
-> 인터넷이 연결되지 않은 환경에서 버전관리가 가능

eGit : eclipse git plugin

1. 설치
 Help -> Install new software -> add
http://download.eclipse.org/egit/updates


2. 로컬 사용법
1) 환경변수 설정
 'HOME'의 이름을 가진 패스를 만든다. : git가 사용되는 기본 패스이다.
*필수로 설정해야 한다.

-> window -> preferences -> team -> git 에서 확인가능

2) 모든 저장소의 기본값으로 사용될 전역설정값 세팅
-> window -> preferences -> team -> git -> configuration
*키값는 user.name 과 같은 형태로 작성
*설정내용은 HOME 패스의 .gitconfig 파일에 저장
3) 프로젝트 생성
4) 프로젝트 쉐어
 team -> shere -> git 선택
 Repository 생성 및 선택

5) Commit
 1)) team -> add
 2)) team -> commit

6) Branch 생성 (subversion)
 team -> switch to -> new branch
* branch가 생성되면 이전 버전과는 별개로 취급된다.
* branch 된 상태에서 add, commit

7) branch 변경
 team -> switch to -> 바꾸고 싶은 버젼

8) history 확인
 team -> show history

9) master 위주로 branch 머지
 1)) Master로 switch
 2)) team -> merge -> branch 선택
 3)) add 후 commit
* branch 에서도 Master 머지 :: 동등해진다

10) branch 삭제
 team -> advance -> delete branch
